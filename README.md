# Personal Portfolio

A responsive **personal portfolio website** for showcasing skills, experience, and contact details.  
Built using **HTML, CSS, and JavaScript**, with a modern dark theme.

## Features
- **Home Section:** Introduction with animated header.
- **About Section:** Detailed description, profile picture, and tabs for **Skills, Experience, and Education**.
- **Contact Section:** Email, phone, and LinkedIn links.
- **Responsive Design:** Works on desktop and mobile.
- **Interactive Tabs:** Switch between Skills, Experience, and Education dynamically.

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Design:** Flexbox, Grid, Responsive Layouts
- **Fonts & Icons:** Google Fonts (Poppins), Font Awesome (optional)
